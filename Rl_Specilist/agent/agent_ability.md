agent 所需要具备的能力：
1. 对逻辑严谨性的感知。 解决问题过程中， 能保证思路和方法的正确性， 不会出现逻辑错误， 并且整体解决问题的逻辑是完整的。
2. 韧性。 对于调用工具失败的情况， 会多尝试几次。
3. 灵活性问题， 对于某种工具调用， 或某种路径已经产生了死胡同时， 能够换一种方式。
4. 选择合适的工具调用。 知道哪种类型问题调用哪种工具。
5. 解决问题的方式。 先大致了解一件事情后， 再开始做事情， 然后边做边修正。
6. 效率问题。 合理分配好自身内心世界模型推演和工具调用的频次， 平衡好效率和客观性。
7. 具体的业务能力。 比如写代码、做表格、 做ppt，具体的技能。
8. 长程目标一致性。 能够在工作很久之后， 仍然记住之前的目标， 并且清晰了解当前所处的位置。
9. 拆解复杂任务能力。 能够把一个复杂的规划， 拆分成N个小点， 然后再逐步完成
10. 对全局规划的能力， 不只是局限于局部进行规划和解决问题， 要有全局意识。
11. 能够坚定事实的能力 。 用户发送错误认知和指令时， 要能够坚持自己， 纠正用户。
